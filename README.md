# TypeScript란?

TypeScript는 Microsoft에서 개발한 오픈 소스 프로그래밍 언어로, JavaScript에 정적 타입을 추가하여 코드의 가독성과 유지보수성을 향상시키는 것을 목표로 합니다. TypeScript는 JavaScript의 상위 집합이며, JavaScript와의 호환성을 지원합니다.
